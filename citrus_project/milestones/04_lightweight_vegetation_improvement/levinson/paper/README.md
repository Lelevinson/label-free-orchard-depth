# Paper build folder — Levinson Milestone 4

A self-contained, Overleaf-ready LaTeX paper drafted from the workspace's existing
results. Hero method = Snapshot 10 (label-free EMA in-domain self-teacher);
Snapshots 00–11 appear as honest ablations / negative results.

## How to build (Overleaf)
1. Upload this whole `paper/` folder to a new Overleaf project (drag-drop the folder,
   or zip it first).
2. Set **main.tex** as the main document. Compiler: **pdfLaTeX**. Bibliography: **BibTeX**.
3. Compile twice (pdfLaTeX → BibTeX → pdfLaTeX → pdfLaTeX) so citations resolve.

No local LaTeX toolchain was available during drafting, so the document was written
carefully but **not compiled here** — skim the first compile for float placement and
overfull-box warnings (the wide qualitative figure spans both columns by design).

## Files
- `main.tex` — the entire paper (single file, per request; tables are inline).
- `references.bib` — bibliography. Entries marked `% UNVERIFIED` need checking.
- `figures/` — all figures used (PNG). Generated by `make_figures.py` + copied panels.
- `make_figures.py` — regenerates the data-driven figures and prints the verified
  numbers from on-disk evaluation artifacts (no GPU, no model inference).
- `OUTLINE.md` — the section/figure/table plan the draft was reviewed against.
- `tables/` — intentionally empty (tables are inline in `main.tex`).

## Numbers provenance (honesty)
Every metric in the paper is read from on-disk evaluation summary JSONs / per-sample
CSVs (Original Lite-Mono milestone-01 results; B0/S05/S07/S09/S10 snapshot results;
S11 gate evals). The hero S10 test result was independently re-run and reproduced
exactly. Training is label-free (RGB-only); LiDAR depth is evaluation-only; inference
is single-image RGB-only Lite-Mono with no architectural change.

## Honesty guardrails enforced in the text
- S10 beats the **original** Lite-Mono on both median-scaled abs_rel and δ1, but B0 and
  S07 have **higher δ1** — the paper says so and never claims best-δ1.
- The DC∧GC reliability gates were **near-inert (~88% kept)**; the paper credits the gain
  to near-dense distillation and calls for an ablation, not to gating.
- S10 improves **metrics, not visuals**; sky/canopy structure remains unsolved.
- Evidence is **single-domain** (CitrusFarm Seq01); labels are project-generated, eval-only.
